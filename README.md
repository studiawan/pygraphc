# pygraphc
Event log clustering tool using Python.

## Requirements
* Python 2.7.6
* NetworkX 1.9.1
* scikit-learn 0.14.1
* nltk 2.0b9
* numpy 1.8.2
* Gephi 0.8.2
* Sphinx 1.3.1
* numpydoc 0.6.0
